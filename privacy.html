Политика конфиденциальности

Действует с 31 мая 2026 г.

1. Общие положения

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок обработки и защиты персональных данных пользователей сайта консультанта по стилю Светланы Подольской (далее — Сайт).

Оператор персональных данных: Светлана Подольская. Контакт для обращений по вопросам персональных данных: svpodols@mail.ru.

Используя Сайт и отправляя заявку через форму записи, вы подтверждаете, что ознакомились с настоящей Политикой.

2. Какие данные мы собираем

При заполнении формы «Запись на консультацию» вы можете передать следующие данные:

  • имя;
  • адрес электронной почты или номер телефона (на ваш выбор);
  • выбранная услуга;
  • дополнительные пожелания (если вы их указали);
  • согласие на обработку персональных данных.

Сайт не собирает специальные категории персональных данных и не предназначен для лиц младше 18 лет без согласия законных представителей.

3. Цели обработки данных

Персональные данные обрабатываются в следующих целях:

  • обработка заявки на консультацию и обратная связь с вами;
  • подбор формата консультации и ответы на ваши вопросы;
  • ведение переписки по выбранному вами способу связи.

4. Правовые основания

Обработка персональных данных осуществляется на основании:

  • вашего согласия, выраженного при отправке формы на Сайте;
  • необходимости действий до заключения договора по вашей инициативе (обработка заявки).

5. Передача данных третьим лицам

Для доставки заявки используются следующие сервисы:

  • FormSubmit — для отправки заявки на электронную почту оператора;
  • Telegram Bot API — для дублирования уведомления о заявке оператору.

Эти сервисы получают только данные, необходимые для передачи заявки. Оператор не продаёт и не передаёт персональные данные третьим лицам в маркетинговых целях.

6. Срок хранения данных

Данные из заявок хранятся не дольше, чем это необходимо для связи с вами, оказания консультационных услуг и выполнения требований законодательства. По вашему запросу данные могут быть удалены досрочно, если их хранение не требуется по закону.

7. Ваши права

Вы вправе:

  • запросить информацию об обработке ваших персональных данных;
  • потребовать уточнения, блокирования или удаления данных;
  • отозвать согласие на обработку персональных данных, направив запрос на svpodols@mail.ru.

Отзыв согласия не влияет на законность обработки, которая была проведена до момента отзыва.

8. Защита данных

Оператор принимает разумные организационные и технические меры для защиты персональных данных от неправомерного доступа, изменения, раскрытия или уничтожения.

9. Файлы cookie и аналитика

Сайт не использует системы веб-аналитики и рекламные трекеры. Могут применяться технически необходимые механизмы браузера для корректной работы отдельных элементов интерфейса.

10. Изменения Политики

Оператор вправе обновлять настоящую Политику. Актуальная версия всегда доступна на этой странице. Продолжая пользоваться Сайтом после публикации изменений, вы соглашаетесь с обновлённой редакцией.

11. Контакты

По вопросам, связанным с персональными данными, вы можете связаться с оператором:

  • электронная почта: svpodols@mail.ru;
  • Telegram: @swetlanapodolskaya.
